# Project Context

- **Name:** ShopERP
- **Type:** Mini Retail Back-Office System (no POS)
- **Target:** Single retail shop
- **Currency:** BDT (Bangladeshi Taka). Smallest unit: paisa (1 BDT = 100 paisa)
- **Language:** English (UI and API)
- **Timezone:** Asia/Dhaka (UTC+6)

## Current State

- **Phase:** Not started
- **Last Sprint:** None

## Environment

- **Backend:** Python 3.12 + FastAPI, port 8000
- **Frontend:** React + Vite, port 5173
- **Database:** PostgreSQL 16, port 5432
- **Docker:** docker-compose for local dev

## Display Conventions

- Prices: ৳ X,XXX.XX (Taka with 2 decimal places, formatted from paisa)
- Dates in UI: DD MMM YYYY (15 Jan 2024)
- Dates in API: ISO 8601 (2024-01-15T10:30:00Z)
- Pagination default: 20 items per page
- Search: case-insensitive, partial match
